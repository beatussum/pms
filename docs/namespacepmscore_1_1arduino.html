◆ set_main_encoders()

void pmscore::arduino::set_main_encoders ( const encoders &  __e)
noexcept

Afin qu'un encoder puisse correctement monitoré le moteur auquel il est associé (afin que le membre m_incrementation est incrémenté à chaque tour), il doit être défini en tant qu'encodeur principal.

Paramètres
__eL'objet encoders regroupant les deux encoders à spécifier en tant qu'encodeurs principaux.

◆ main_encoders

encoder* pmscore::arduino::main_encoders[] = {nullptr, nullptr}
inline

Tableau contenant deux pointeurs pointant vers les deux encodeurs principaux.

Les deux encodeurs principaux sont les deux encodeurs dont la position est effectivement monitorée par le programme. Cette implémentation s'explique par les limitations imposées par les « interrupt handler functions ».

Le premier élément correspond à l'encodeur A (celui de gauche) et le deuxième élément à l'encodeur B (celui de droite).

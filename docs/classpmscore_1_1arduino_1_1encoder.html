◆ reverse()

void pmscore::arduino::encoder::reverse ( )
inlinenoexcept

Active le mode de comptage inverse de l'encodeur.

Lorsque le mode de comptage inverse est activé, le moteur monitoré par l'encodeur est considéré comme tournant dans le sens négatif. En d'autres termes, le membre m_incrementation est décrémenté (au lieu d'être incrémenté) de 1.

friend

Spécifie deux encoder en tant qu'encodeurs principaux.

Afin qu'un encoder puisse correctement monitoré le moteur auquel il est associé (afin que le membre m_incrementation est incrémenté à chaque tour), il doit être défini en tant qu'encodeur principal.

Paramètres
__aL'encodeur A (associé à la roue gauche).
__bL'encodeur B (associé à la roue droite).
